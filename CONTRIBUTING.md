# 기여 안내 / Contributing

## 한국어

이 저장소는 PPS CLI의 공개 배포 저장소이며 애플리케이션 소스 코드를 포함하지 않습니다. 다음 기여를 환영합니다.

- 재현 절차가 포함된 버그 보고
- 설치 문서와 번역 개선
- 패키지 관리자 메타데이터 수정
- 지원 운영체제에서의 호환성 보고

이슈를 만들기 전에 최신 릴리즈에서 문제가 재현되는지 확인하고, 운영체제·아키텍처·`pps --version` 결과를 포함해 주세요. 보안 취약점은 공개 이슈가 아니라 [SECURITY.md](SECURITY.md)의 비공개 절차로 신고해야 합니다.

애플리케이션 기능 패치는 공개 저장소에서 직접 받지 않습니다. 기능 제안은 이슈로 제출하면 내부 개발 저장소에서 검토합니다.

## English

This is the public distribution repository for PPS CLI and does not contain the application source. Contributions are welcome for reproducible bug reports, documentation and translation improvements, package metadata, and platform compatibility reports.

Before opening an issue, reproduce it with the latest release and include your operating system, architecture, and `pps --version` output. Report vulnerabilities privately according to [SECURITY.md](SECURITY.md).

Application feature patches cannot be accepted directly in this repository. Submit feature proposals as issues for review in the private development repository.
