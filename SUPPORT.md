# 지원 / Support

## 한국어

1. `pps --version`으로 최신 버전인지 확인합니다.
2. `pps <command> --help`에서 명령 형식을 확인합니다.
3. [설치 안내](docs/installation.md)와 기존 [이슈](https://github.com/PROJECT-PS/PPS-CLI/issues)를 검색합니다.
4. 해결되지 않으면 이슈 템플릿을 사용해 새 이슈를 만듭니다.

이슈에는 운영체제, 아키텍처, CLI 버전, 실행한 명령, 실제 출력과 기대 결과를 포함해 주세요. `~/.pps/config.json` 또는 `%APPDATA%\PPS\config.json`의 토큰을 절대로 첨부하지 마세요.

## English

Confirm the version with `pps --version`, review `pps <command> --help`, search the [installation guide](docs/installation.en.md) and existing [issues](https://github.com/PROJECT-PS/PPS-CLI/issues), then open a new issue with the provided template.

Include your operating system, architecture, CLI version, command, actual output, and expected result. Never attach the token stored in `~/.pps/config.json` or `%APPDATA%\PPS\config.json`.
